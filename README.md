### Пользовательский скрипт для Perfect World

Панель для выбора персонажей, управления подарками и автоматической передачи предметов в Perfect World.

### Основные возможности

* **Автовыбор персонажа** - автоматический выбор персонажа при загрузке страницы
* **Группы предметов** - возможность создавать и редактировать группы предметов
* **AJAX-передача** - передача предметов без переадресации страницы
* **Последовательная передача** - автоматическая передача предметов выбранной группы пакетами по 5 штук (рекомендуется для передачи большого количества подарков)
* **Открытие сундуков** - автоматическое открытие сундуков караванщика

## 

Скрипт работает только на странице подарков Perfect World

* https://pwonline.ru/promo_items.php
* https://pwonline.ru/promo_items.php?do=activate&cart_id=*

Все настройки сохраняются локально в браузере

### Установка

[Ссылка на скрипт](https://github.com/abc9702/pw-gifts-panel/raw/refs/heads/main/pw-gifts-panel-enhanced.user.js)

Для запуска скрипта установите расширение (например, Tampermonkey).


Включите в настройках расширений режим разработчика и разрешите пользовательские скрипты в настройках расширения.
